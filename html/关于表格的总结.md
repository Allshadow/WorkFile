### 基本表格结构

以下为最简单的表格结构

​	ps :  \<th> 标签默认居中

index.html

```
<table>
  <tr>
    <th>Month</th>
    <th>Savings</th>
  </tr>
  <tr>
    <td>January</td>
    <td>$100</td>
  </tr>
</table>
```

index.css

```
<style>
	table{
		width: 100%; //将表格设为自适应宽度
	}
</style>
```
