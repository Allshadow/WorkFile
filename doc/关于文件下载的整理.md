##  a标签下载

1）链接直接下载

```
//<a>标签直接下载
<a href="https://www.juejin.com/测试.xls" download="测试.xls">EXCEL</a>

//动态<a>标签下载
let a = document.createElement("a");
a.href = "http://www.juejin.com/测试.xls"
a.download = '测试.xls'
//以下这段为了兼容火狐
a.dispatchEvent(new MouseEvent('click', {bubbles: true, cancelable: true, view: window}))
a = null;
```
