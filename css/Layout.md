## 响应式布局

### 简介

响应式布局通常分为三种：

- 利用弹性盒子 Flex、网格布局 Grid 或左浮动法 Float 等 css 属性来实现的在一列上根据屏幕大小不同而进行自动换行
- 利用媒体查询来根据当前屏幕大小来运行相应的css代码
- 利用 javascript 来获取窗口宽高来动态操作 DOM 元素
